From Script to Sound in Three Simple Steps

No complex setup required. Get started in minutes.

Write Your Script

Create a simple text file with your content. You can even add tags like (intro_music) or (transition_sound) for automatic assembly.

# podcast_script.txt
Host: Welcome to AI Spotlight!
Guest: Thanks for having me!
(transition_sound)
Host: Let's discuss AI...
(outro_music)

Choose Voices

Select different voice models for each speaker. Create natural conversations with distinct voices for hosts, guests, and narrators.

$ ls voices/
host_sarah_en.onnx
guest_alex_en.onnx
narrator_emma_en.onnx

Generate Podcast

Run the script and watch as your text transforms into a natural conversation between multiple speakers with perfect timing and intonation.

$ python podcast_ai.py --script conversation.txt --voices host_sarah,guest_alex
# Generating host voice...
# Generating guest voice...
# Mixing conversation...
# Done! Saved to podcast.mp3
